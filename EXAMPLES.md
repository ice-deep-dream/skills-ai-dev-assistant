```typescript
// 修复前
const captchaUrl = `/captcha?id=${id}`;

// 修复后
const captchaUrl = `${API_BASE_URL}/captcha?id=${id}`;

## 关键要点总结

### ✅ 必须遵守的规则

1. **每次对话前调用 skill** - 确保文档与开发同步
2. **所有文档写入 docs 目录** - 统一管理
3. **按照模板格式编写** - 保持规范性
4. **记录真实状态** - 不写空泛描述
5. **文档与代码一致** - 及时更新

### 📝 文档类型对应关系
