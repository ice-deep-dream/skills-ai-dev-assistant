# 【模块名称】后端模块开发文档

> ⚠️ **核心原则：模板服务于项目，项目优先于模板**
> 
> **当模板与项目冲突时，必须修改模板适配项目，绝不允许让项目适配模板！**
> 
> 💡 **使用说明**：
> 
> 1. **第一步：项目分析** - 运行技能分析现有项目结构
> 2. **第二步：识别差异** - 对比模板与项目的差异点
> 3. **第三步：修改模板** - 根据项目特点调整模板内容
> 4. **第四步：生成文档** - 使用适配后的模板生成文档
> 
> 🚨 **强制规则**：
> - ❌ 禁止直接套用模板
> - ❌ 禁止修改现有代码风格适配模板
> - ✅ 必须修改模板适配现有项目
> - ✅ 必须保持与现有代码的一致性

## 四、实体定义（动态适配）

### 4.1 根据 ORM 选择

**【根据项目使用的 ORM，选择以下一种写法】**

#### 选项 A：TypeORM（推荐）

```typescript
// modules/【模块名】/entities/【实体】.entity.ts
import {
  Entity,
  PrimaryGeneratedColumn,
  Column,
  CreateDateColumn,
  UpdateDateColumn,
  Index,
  DeleteDateColumn,
} from 'typeorm'

export enum 【实体】Status {
  ACTIVE = 'active',
  INACTIVE = 'inactive',
  PENDING = 'pending',
}

@Entity('【表名】')
@Index(['name'], { unique: true })
export class 【实体】 {
  @PrimaryGeneratedColumn('uuid')
  id: string

  @Column({ length: 100 })
  name: string

  @Column({ type: 'text', nullable: true })
  description: string

  @Column({
    type: 'enum',
    enum: 【实体】Status,
    default: 【实体】Status.ACTIVE,
  })
  status: 【实体】Status

  @Column({ type: 'jsonb', nullable: true })
  metadata: Record<string, any>

  @CreateDateColumn()
  createdAt: Date

  @UpdateDateColumn()
  updatedAt: Date

  @DeleteDateColumn()
  deletedAt: Date

  @Column({ nullable: true })
  createdBy: string

  @Column({ nullable: true })
  updatedBy: string
}
```

## 七、Service 开发

### 7.1 标准 Service 实现

```typescript
// modules/【模块名】/【模块名】.service.ts
import { Injectable, NotFoundException, BadRequestException } from '@nestjs/common'
import { InjectRepository } from '@nestjs/typeorm'
import { Repository } from 'typeorm'
import { 【实体】 } from './entities/【实体】.entity'
import { Create【实体】Dto } from './dto/create-【实体】.dto'
import { Update【实体】Dto } from './dto/update-【实体】.dto'

@Injectable()
export class 【模块名】Service {
  constructor(
    @InjectRepository(【实体】)
    private readonly repository: Repository<【实体】>,
  ) {}

  async findAll(params: {
    page: number
    pageSize: number
    keyword?: string
  }) {
    const { page, pageSize, keyword } = params
    
    const query = this.repository.createQueryBuilder('entity')
    
    if (keyword) {
      query.where('entity.name LIKE :keyword', { keyword: `%${keyword}%` })
    }
    
    const [items, total] = await query
      .orderBy('entity.createdAt', 'DESC')
      .skip((page - 1) * pageSize)
      .take(pageSize)
      .getManyAndCount()
    
    return {
      list: items,
      total,
      page,
      pageSize,
    }
  }

  async findOne(id: string) {
    const entity = await this.repository.findOne({ where: { id } })
    
    if (!entity) {
      throw new NotFoundException('资源不存在')
    }
    
    return entity
  }

  async create(dto: Create【实体】Dto) {
    const entity = this.repository.create(dto)
    return await this.repository.save(entity)
  }

  async update(id: string, dto: Update【实体】Dto) {
    const entity = await this.findOne(id)
    
    Object.assign(entity, dto)
    return await this.repository.save(entity)
  }

  async remove(id: string) {
    const entity = await this.findOne(id)
    await this.repository.remove(entity)
  }
}
```
