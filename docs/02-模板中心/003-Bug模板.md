```
[新建] → [已分配] → [处理中] → [待验证] → [已解决] → [已关闭]
  ↓         ↓          ↓          ↓          ↓
[已拒绝]  [重新分配]  [挂起]    [重新打开]
```

## 填写说明

1. **标题**：简洁描述问题，包含关键词便于搜索
2. **优先级**：根据影响范围和紧急程度选择
3. **严重级别**：根据问题对系统的影响选择
4. **复现步骤**：尽量详细，确保他人能够重现
5. **环境信息**：尽可能完整，便于定位问题
6. **附件**：提供截图、日志等有助于诊断的材料
